#  Forge – Build Real Skills, Not Just Watch Tutorials

Forge is a modern, responsive landing page designed for aspiring developers who want to learn through real-world projects instead of only watching tutorials.

As part of my internship, I enhanced the landing page by integrating a **Project Dashboard (Build Tracker)** using **Vanilla JavaScript**, allowing users to manage project tasks dynamically.

---

##  Internship Task

**Task 2 – To-Do List Web Application**

This project demonstrates the implementation of a dynamic task management system using **HTML, CSS, and Vanilla JavaScript** without any external frameworks.

---

##  Features

### Landing Page
- Modern and responsive UI
- Hero section with call-to-action
- Learning tracks section
- How It Works section
- Outcomes/Testimonial section
- Responsive navigation bar
- Professional footer

### Project Dashboard
-  Add new tasks
-  Mark tasks as completed
-  Delete individual tasks
-  Live task statistics
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
-  Dynamic progress bar
-  Clear all completed tasks
-  Add tasks using the Enter key
-  Fully responsive layout

1. Download or clone the repository.

```bash

```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or additional packages are required.
